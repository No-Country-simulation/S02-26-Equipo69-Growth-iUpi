# Métricas e hipótesis – iUpi MVP

Este documento define qué se mide y por qué.

## Hipótesis principales

1. Si humanizamos la comunicación, aumenta la confianza.
2. Si mostramos el simulador en uso, baja la fricción.
3. Si aclaramos el CTA, mejora la conversión.
4. Si medimos el canal de origen, optimizamos adquisición.


## Métricas clave

- Conversion Rate (visitas → registros).
- Registros por canal.
- CTR desde redes.
- Abandono del formulario.
- Evolución antes / después.


## Uso de métricas

Las métricas no se usan para “reportar números”,
sino para aprender y tomar decisiones.

El MVP existe para validar, no para escalar.
